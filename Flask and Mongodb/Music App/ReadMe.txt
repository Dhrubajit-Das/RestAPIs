Refer database schema (image) for the structure of the database.

Operations:

1. Insert
2. Search
3. Delete

a. Insert:
  i. Insert into Artist
 ii. Insert into Album
iii. Insert into Track


b. Search:
  i. Artist: Search artist by name. Fetch count of albums for that artist.
 ii. Album : Search album by name. Fetch artist details and count of tracks for that album.
iii. Track : Search track by name. Fetch album details and artist details for that track.


c. Delete:
  i. Artist: Delete artist details --> delete all albums for artist --> delete tracks for all those albums.
 ii. Album : Delete album details  --> delete all tracks for that album
iii. Track : Delete track details.
